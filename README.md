# ⚔️ Terminal Legends

> A terminal-based RPG where players fight monsters, level up, and explore a simple game world.

---

## 📖 About the Project

**Terminal Legends** is a text-based role-playing game (RPG) built with Python.  
The game runs in the terminal and allows players to create a character, battle enemies, gain experience, and progress through levels.

---

## 🚀 Features

- 🧙 Character creation (name + class)
- ⚔️ Turn-based combat system
- 🐉 Different enemy encounters
- 📈 Leveling system (XP)
- 💻 Fully playable in the terminal

---
